What is DOM?

The DOM (Document Object Model) is a programming interface for web documents, representing the structure of a webpage as a tree of objects (nodes) that can be manipulated with languages like JavaScript. It allows developers to dynamically modify the content, structure, and styling of a webpage in response to user actions or other events.

How do you select an element from dom?

You can select an element from the DOM using JavaScript methods like document.getElementById('id') to get an element by its ID or document.querySelector('selector') for selecting the first element that matches a CSS selector.

What is event delegation in the context of dom and why is it useful?

Event delegation is a technique in the DOM where you attach a single event listener to a parent element instead of individual child elements, allowing the event to "bubble up" from the child to the parent. This is useful because it reduces memory usage, simplifies code (especially when dealing with dynamically added elements), and improves performance by minimizing the number of event listeners needed.

How do you manipulate an elements attribute and style using the DOM?

You can manipulate an element's attribute using setAttribute() or getAttribute(), and modify its style by accessing the style property in the DOM.
